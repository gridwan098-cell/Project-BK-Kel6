📖 Deskripsi

Program ini merupakan aplikasi kalkulator sederhana berbasis Command Line Interface (CLI) yang dibuat menggunakan bahasa pemrograman Go. Program ini memungkinkan pengguna untuk melakukan operasi aritmatika dasar seperti penjumlahan, pengurangan, perkalian, dan pembagian.

Program akan meminta pengguna memasukkan:

1. Angka pertama

2. Operator (+, -, *, /)

3. Angka kedua

Kemudian program akan menampilkan hasil perhitungan dengan dua angka di belakang koma.

⚙️ Fitur

✅ Penjumlahan (+)

✅ Pengurangan (-)

✅ Perkalian (*)

✅ Pembagian (/)

✅ Validasi pembagian dengan nol

✅ Validasi operator tidak valid
